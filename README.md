# 0.i 大地图唯一地点解锁

这是一个用于《大灾变：黑暗之日》(Cataclysm: Dark Days Ahead, CDDA) 0.i 版本的本地小型平衡/测试模组。

## 功能说明

本模组会将部分原版中 `GLOBALLY_UNIQUE` 的大地图特殊地点改为 `OVERMAP_UNIQUE`。这样这些地点不再是整个世界全局唯一，而是按大地图区域保持唯一生成。

当前调整的地点包括：

- Hub 01
- 难民中心
- 无家者营地
- LIXA
- Exodii 基地

## 使用场景

这个模组主要用于稳定性测试或长档游玩时的地图生成调整，避免部分关键唯一地点因为世界生成、探索范围或特殊生成条件导致难以找到。

它不会新增地图内容，也不会改动任务逻辑；核心变化只是放宽上述特殊地点的唯一性限制。

## 安装方式

1. 将整个 `overmap_unique_unlocks` 文件夹放入 CDDA 的 `data/mods/` 目录。
2. 创建世界或编辑世界模组列表时启用 `0.i 大地图唯一地点解锁`。
3. 启动游戏并正常生成/探索世界。

## 注意事项

- 模组 ID 保持为 `overmap_unique_unlocks`，请不要随意修改，否则已有世界的模组引用可能失效。
- 对已经生成过的大地图区域，本模组通常不会 retroactively 重刷既有地形；效果主要体现在新的世界或尚未生成的区域。
- 如果更新模组后遇到 `stale game data` 一类缓存问题，可以尝试清理游戏目录下的 `cache/` 和 `data/cache/` 后再进游戏。
